Jugsaw

Jugsaw is a toolkit designed to assist scientific computing scientists in deploying their applications to the cloud. Applications are deployed on the Jugsaw website as Jugsaw apps, which consist of a set of functions.

Note

Currently, these functions must be written in Julia or encapsulated within a Julia wrapper (C, Fortran, Python).
